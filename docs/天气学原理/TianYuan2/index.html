气团和锋

气团​

  • 气团的概念：气团是指气象要素（==主要是指温度和湿度==）水平分布比较均匀的大范围的空气团。在同一个气团中，各地的气象要素的垂直分布（稳定度），几乎相同，天气现象也大致一样。气团的水平尺度可达几千千米，垂直范围可达几千米到十几千米，常常从地面伸展到对流层顶。
  • 气团的分类
    1. 地理分类法：
      • 北极气团
      • 极地气团
      • 热带气团
      • 赤道气团
    2. 热力分类法：
      • 暖气团
      • 冷气团

锋​

锋的概念​

  • 锋：冷暖气团相遇，存在一个狭窄的过度带，此过渡带，此过渡带向冷区倾斜，称此过渡带为锋

  • 锋面：在近地面过渡带宽为约数十公里，在高层可达 200-400km，其宽度与长度相比（数千公里）是很小的，可以近似看作一个面，即锋面。

    靠近暖区叫锋的上界，靠近冷区叫锋的下界。

  • 锋区：锋面与空中平面相交的区域称为锋区

  • 锋线：锋面与地面的交线

锋的分类​

  • 冷锋：冷气团起主导作用，推动锋面向暖气团一侧移动，称为冷锋
  • 暖锋：暖气团起主导作用，推动锋面向冷气团一侧移动，称为暖锋
  • 准静止锋：冷暖气团势力相当，锋面很少移动的锋，称为准静止锋（6 小时位置变化不大，24 小时在 2 个维度以内）
  • 锢囚锋：暖气团，较冷气团，更冷气团相遇时，先构成两个锋面，然后其中一个追上另外一个锋面即形成古囚。

锋面附近的气象要素场的特征

温度场特征​

  • 从垂直剖面图看：

    1. 锋区内==水平温度梯度==比同高度上两个气团内大
    2. 锋区内==垂直温度梯度==比锋区外气团小

    ​

  • 从探空曲线上看：锋区存在时，其探空曲线有三种类型：（1）逆温层（2）等温层（3）很小的垂直递减率

  • 从等压面上看：

    1. 如果用 850，700hPa 等压面去截取锋面，则

      (1) 等压面上锋区内等温线较两侧密集

      (2) 等压面上锋区内等温线走向与地面锋线基本平行

    2. 如果把每层等压面上锋区位置移到同一张图上，则会发现，越到高层，锋区位置越偏向与冷空气一侧。

    3. 高空上锋区附近的冷暖平流与锋区性质的关系

      冷锋对应==冷平流==，暖锋对应==暖平流==

      若无平流或仅有弱的冷暖平流，则对应静止锋

    4. 锢囚锋在等压面图上的表现形式——有==暖舌==

气压场特征​

  • 在垂直与锋面的方向上，锋面两侧的气压梯度是不连续的，等压线通过锋面时必然发生弯折，且折角指向高压

风场特征​

  • 锋面附近的风场具有==气旋式切变==
  • 锋面附近有摩擦==辐合==与==上升==运动
  • 锋区上空存在急流：极锋锋区和副热带锋区存在西风急流

变压场特征​

  • 变压特征：冷锋后ΔP>0\Delta P>0ΔP>0,暖锋前ΔP<0\Delta P<0ΔP<0

  • 锋线移动速度大小和变压梯度大小成正比

湿度场特征​

  • 一般情况：暖空气湿度大；冷空气湿度小
  • 锋面两侧的湿度差大（Td 差大）

锋面天气​

  • 冷锋附近的云和降水

  • 暖锋附近的云的降水

    特征：锋线位于高空槽前，坡度小移速慢冷暖气团都为上升气流；以层状云为主，从高到低垂直排列，锋前稳定降水

  • 准静止锋附近的云和降水

    特征：类似于第一类冷锋，但坡度更小，垂直运动弱，以层状云为主；稳定的连续降水，雨量小，锋区宽，时间长，雨区立锋线有一段距离。

锋生和锋消

  • 锋生：指密度不连续性形成的一种过程或已经有的一条锋面，其温度或位温水平梯度加大的过程。
  • 锋消：锋生相反的过程
  • 锋生区域：集中在华南到长江流域和河西走廊到东北
  • 锋消区域：主要在青藏高原以东 30°-40°N 一带（地形原因）。
