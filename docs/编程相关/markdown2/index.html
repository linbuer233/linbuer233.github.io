上标、下标和脚注

上标的语法为

^+数字+^

例：x^2^

下标的语法为

~+数字+~

例：x2

脚注是

文字 [^数字]
[^ 数字]: 内容

例： 哈哈哈1

更改颜色

有三种方式去设置颜色：

  1. 直接用英文名 如Red
  2. 用RGB三原色 如rgb(0,0,0) 为黑色
  3. 用十六进制表示法 如#000000(黑色)

语法格式：

$\color{设置颜色}{文本内容}$

例： $\color{red}{绿色}$

绿色\color{red}{绿色}绿色

$\color{rgb(0,0,0)}{白色}$

\color{rgb(0,0,0)}{白色}

$\color{#008000}{白色}$

白色\color{#008000}{白色}白色
